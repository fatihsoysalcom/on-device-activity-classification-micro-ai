# On-Device Activity Classification Micro-AI

This example simulates a 'Micro-AI' application running on a resource-constrained consumer device, such as a fitness tracker. It demonstrates how sensor data is processed locally using a lightweight model to classify activity levels (sedentary, walking, running) without sending any data to a cloud server. This approach ensures low latency, enhanced data privacy, and reduced operational costs, embodying the core principles of Micro-AI engineering.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
